# MarianMoodleMate 🎓

MarianMoodleMate is a Chrome extension that enhances the Moodle experience by providing a clean, persistent dashboard for assignments and courses with smart reminders and quick navigation.

---

## 🚀 Features

### 📌 Assignments Dashboard
- View all assignments in one place
- Time remaining and progress indicators
- Persistent across navigation and reloads

### 📚 Course Navigation
- Access all courses instantly
- Inline rename feature for personalization

### 🔄 Sync System
- Manual refresh for instant updates
- Auto-sync every 30 minutes
- Accurate "Last Synced" tracking

### 🔔 Smart Notifications
- Alerts for upcoming and urgent assignments
- Anti-duplicate logic to prevent spam

### 🎨 Premium UI
- Floating assistant accessible on all Moodle pages
- Clean, modern interface with smooth interactions
- Minimal scrollbars and responsive design

---

## 🧠 Tech Stack

- JavaScript (Vanilla)
- Chrome Extension APIs (Manifest V3)
- chrome.storage.local
- chrome.notifications

---

## 🔐 Privacy

- No external servers
- No data collection
- Works entirely within the browser

---

## ⚙️ Installation

1. Download or clone this repository
2. Open Chrome → `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select the extension folder
